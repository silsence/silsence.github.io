1. scanf不能写出%.1lf,会发生错误,应该写成%lf;

2. 取模并倒转,结果由除数决定,例 : -123%10=-321,///,-123%-10=-321,///,-380%10=-83,///,380%-10=83;

4. c语言中,%s用来输入字符串 : scanf("%s",&name);

5.  程序运行前，只有代码区和全局区，程序运行后，有堆区和栈区
    | 名称   | 作用  |
    | :---- | :----- |
    | 代码区 | 存放函数体内的二进制代码，有操作系统管理 |
    | 全局区 | 存放全局变量，静态变量以及常量 |
    | 堆区   | 如malloc,由程序员分配释放，若程序员未释放，则程序结束之后，编译器自动释放 |
    | 栈区   | 由编译器自动分配和释放，存放函数参数，局部变量等 |

6. a=a&(a-1)将a的最右位变为0;

7. "|"把末尾的最后一位变为1,如果要为0就减去1

8. 异或' ^ ',相同为0,不同为1,a^b=(a AND (NOT b)) OR ((NOT a) AND b)

9. 取反' ~ ',每一位都取反,包括符号位取反,

10. 左移' << ',a<< b,把a左移b位，相当于a乘2的b次方,a<<1相当于乘了2

11. 右移' >> ',a>>b,把a右移b位,相当于a除2的b次方,a>>1相当于除了2,判断正负时,右移31位,可以得到符号位来判断正负,

12. 格雷码，自然二进制数在高位补0,相邻2位之间做或运算,B=0 0101转为G=0111

13. 格雷码还原为二进制码，将格雷码高位补0,相邻之间的数做异或比较,G=0 1010转为B=1100

14. |优先级|名称|运算符|
    | --- | --- | :---: |
    |  1  |按位取反  |   ~   |
    |  2  |乘,除,模 | * / % |
    |  3  |加,减    | + -   |
    |  4  |左移,右移 | << >> |
    |  5  |按位与    |   &   |
    |  6  |异或      |   ^   |
    |  7  |按位或    |   "|" |
